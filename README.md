# Social Node
A social app that allows you to communicate with other app users by writing posts or by chatting.
</br>

<span style="color: #824341">
The website was created for the purpose of learning programming. It may contain bugs.
</span>

</br>

# Reporoduce
<div style="background-color: gray">
    
    git clone https://github.com/Klaudiusz16/socialNode.git

    We need use 2 terminals (for frontend and backend server)

    In backend terminal folder:
      yarn install
      yarn start

    in frontend terminal folder:
      yarn install
      yarn start
  
    * It is good to remove .cache folder if exist. Folder can cause error.
